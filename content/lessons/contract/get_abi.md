---
title: "Get ABI"
---

Fetch the ABI of a given contract. ABI stands for Application Binary Interface. This is the interface necessary to talk to smart contracts.
If the address is not a contract or the contract is not known to us, the call will return with a 404.

## Example Request

```javascript
curl -X GET \
  -H 'Content-Type: application/json' \
  https://api.cinder.cloud/api/ethereum/contract/0xbbeb960fbed8b25cbde5e949e2bdc18c6a58444a/abi
```
